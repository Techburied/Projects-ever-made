# 10 Coding Projects
My greatest project in which I have invested the most amount of time refining it is my artificial assistant - friday. It is inspired by Iron man's friday. This robot is a beautiful combination of raspberry pi and arduino, both works in harmony to provide a seemless output. This is not just a chat bot but it helps with general things such as telling recipies, motivating and much more. since it is integrated with my humanoid robot, this enables friday to interact with real life world and perform specific gestures for specific person with the use of face detection technology. Also I learned from scratch how to build apps to make customised app for this robot whose apk file is still available to download. It took me a whole 2 years to build the robot as well as friday and around 20 - 30k of capital. This project is still being refined as I am still learning more and more on regular basis and I plan to continue working on this project even after years. 

Another big project was bike sales management using pandas and matplotlib. This was the project that I made during my final year at school. This project uses python to ease managing sales of bikes and its models. It comes with various features such as sorting data, accessing inventory and sales, visualising data using matplotlib and much more. It took me around a month to build it on my own and I was awarded with full grades for it.

Majority of the projects were completed in the span of 2 years and some after that. 
These projects are 100% authenticate and solely made by me WITHOUT any use of AI. Some of the projects are simple but are put to show a specific set of logic. For example, dictionary site is simple looking but it is put to show the logic of api use, same with weather website. Below are the outputs of projects:

1. Friday – AI Assistant & Humanoid Robot: An AI voice assistant integrated with my robot which interacts with real world and users through speech, camera vision, and self-made app control via bluetooth.

   Key features:
   1. Voice conversation and questions answering
   2. Collision from object prevention using sensors
   3. Gratitude, motivating and helpful conversation capabilities
   4. Speech recognition
   5. Text-to-speech responses
   6. Weather updates
   7. Dictionary lookup
   8. Custom made app robot control
   9. Bluetooth communication
   10. Camera integration
   11. Real-time interaction

   Key Technologies:
   1. Python Modules such as: SpeechRecognition, gTTS / pyttsx3, Flask, Android, Bluetooth, APIs
   2. Raspberry Pi 3B
   3. Arduino UNO

2. Dictionary Website: This projects shows my skill of word search using dictionary API in JavaScript.
3. Snapchat login clone and twitter clone: this shows my understanding of UI as well as control of html and css skills
4. Live Weather Updates: This project shows my html, css and JS skills of using weather APIs.
5. Bike Sales Management: This shows my data analytical skills in python using Pandas module
6. Payment Forms: This shows my understanding of basics of html and css which are crucial for bigger and advanced projects.
7. Pomodoro with Music - https://techpom.netlify.app/ : This shows my understanding of html, css and JS at advanced levels. Along with using API to integrate yt in it. link to visit it: https://techpom.netlify.app/
